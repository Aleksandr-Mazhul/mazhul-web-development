# Обзор Проектов — JavaScript (Семестр 2)

Данная директория представляет собой коллекцию учебных проектов, демонстрирующих глубокое и уверенное владение
современным "чистым" JavaScript (Vanilla JS), объектно-ориентированным программированием (ООП) и продвинутыми
архитектурными паттернами для создания интерактивных веб-приложений.

Каждый проект в этой директории — это не просто выполненное задание, а полноценное мини-приложение со своей продуманной
архитектурой, инкапсуляцией и управлением состоянием.

## 🚀 Ключевые Проекты и Демонстрируемые Навыки

### 1. **Simon Says (`/SimonSays`)**

Классическая игра на развитие памяти, реализованная с применением профессион��льных техник:

- **Паттерн "Конечный автомат" (State Machine)**: Вместо хаотичных `if/else` и булевых флагов, логика игры управляется
  через строгий набор состояний (`IDLE`, `SHOWING_SEQUENCE`, `WAITING_INPUT`, `GAME_OVER`), что исключает баги и делает
  код предсказуемым.
- **Инкапсуляция (ES6+ Private Fields)**: Внутреннее состояние игры (последовательность, текущий шаг, рекорд) надежно
  защищено с помощью приватных полей (`#sequence`, `#currentStep`), что является золотым стандартом в современном ООП.
- **Асинхронное программирование (`async/await`)**: Анимации и задержки реализованы элегантно и без "Callback Hell",
  демонстрируя понимание событийного цикла (Event Loop) в JavaScript.
- **Паттерн "Делегирование событий"**: Для обработки кликов используется один слушатель на родительском элементе, что
  оптимизирует производительность и потребление памяти.

### 2. **Flashcards (`/flashcards`)**

Полноценное SPA (Single Page Application) для создания и заучивания флеш-карточек. Архитектура этого проекта имитирует
подходы, используемые в больших фреймворках (React, Vue):

- **Модульная структура (ES6 Modules)**: Код строго разделен на логические слои (`models`, `ui`, `app`, `study`), что
  обеспечивает масштабируемость и простоту поддержки.
- **Паттерн "Модель-Представление-Контроллер" (MVC)**:
    - **Модели (`Deck`, `Card`)**: Отвечают за структуру данных.
    - **Представление (`ui/render.js`)**: Набор "чистых" функций, которые рендерят UI на основе полученных данных, но не
      содержат логики.
    - **Контроллер (`App.js`)**: Центральный класс, который управляет состоянием, обрабатывает действия пользователя и
      синхронизирует данные с `localStorage`.
- **Продвинутая работа с `localStorage`**: Реализованы методы `toJSON()` и `fromJSON()`, которые позволяют корректно
  сохранять и восстанавливать экземпляры классов, а не просто "голые" объекты.

### 3. **Game12 (`/game12`)**

Пошаговая настольная игра в кости, демонстрирующая навыки объектной декомпозиции и управления сложной игровой логикой:

- **Объектная модель**: Игра декомпозирована на логические сущности (`Game`, `Player`, `Dice`), каждая из которых
  отвечает за свою часть логики.
- **Разделение логики и представления (Separation of Concerns)**: Классы (`Game`, `Player`) ничего не знают о DOM. Они
  лишь управляют состоянием. За отрисовку отвечают отдельные функции, которые читают данные из экземпляра класса `Game`
  и строят на их основе HTML.
- **Динамический UI**: Интерфейс (кнопки доступных ходов) гене��ируется "на лету" на основе сложной логики, которая
  анализирует бросок костей и доступные карточки игрока.

## 🎯 Общие продемонстрированные компетенции:

- **Уверенное владение ООП**: Классы, наследование, инкапсуляция, полиморфизм.
- **Современный синтаксис ES6+**: Стрелочные функции, `let/const`, деструктуризация, spread/rest-операторы, модули.
- **Асинхронный JavaScript**: `Promise`, `async/await` для управления анимациями и асинхронными операциями.
- **Продвинутая работа с DOM API**: Создание элементов, управление классами, атрибутами и событиями без использования
  jQuery.
- **Архитектурные паттерны**: State Machine, MVC, Event Delegation, Separation of Concerns.
- **Работа с `localStorage`**: Сохранение и восстановление состояния приложений.

Эта коллекция проектов подтверждает способность не просто "писать код", а проектировать и создавать надежные,
масштабируемые и поддерживаемые веб-приложения на чистом JavaScript.
