# Портфолио Учебных Проектов: Веб-разработка и Веб-дизайн

Данный репозиторий представляет собой комплексную коллекцию академических проектов, выполненных в рамках курсов по веб-разработке и веб-дизайну. Работы демонстрируют систематическое освоение технологий от базовых принципов HTML/CSS до продвинутых архитектурных паттернов в JavaScript.

Каждый проект является практическим подтверждением глубоких теоретических знаний и способности применять их для создания современных, интерактивных и визуально привлекательных веб-приложений.

---

## 🚀 Часть 1: Продвинутая JavaScript-разработка (`/javascript-semester-2`)

Этот раздел посвящен созданию интерактивных при��ожений на "чистом" JavaScript (Vanilla JS) с использованием современных стандартов и профессиональных паттернов проектирования. Основной упор сделан на написание поддерживаемого, масштабируемого и надежного кода.

### Ключевые проекты и продемонстрированные навыки:

1.  **`Simon Says`**: Эталонная реализация игры на память, демонстрирующая:
    *   **Паттерн "Конечный автомат" (State Machine)** для безупречного управления состоянием игры.
    *   **Надежную инкапсуляцию** с помощью приватных полей классов (ES6+).
    *   Элегантную обработку анимаций через **`async/await`**.

2.  **`Flashcards`**: Полноценное SPA (Single Page Application), архитектура которого имитирует лучшие практики больших фреймворков:
    *   **Модульная структура (ES6 Modules)** с четким разделением на слои (`models`, `ui`, `app`).
    *   Ре��лизация **паттерна MVC (Модель-Представление-Контроллер)** для разделения данных, логики и отображения.
    *   **Продвинутая работа с `localStorage`** через методы сериализации/десериализации (`toJSON`/`fromJSON`).

3.  **`Game12`**: Пошаговая настольная игра, показывающая мастерство в объектной декомпозиции:
    *   **Четкое разделение логики и представления (Separation of Concerns)**, где классы не зависят от DOM.
    *   **Динамическая генерация UI** на основе вычисляемого состояния игры.

### Общие компетенции в JavaScript:
- **Архитектурные паттерны**: MVC, State Machine, Event Delegation.
- **Объектно-ориентированное программирование (ООП)**: классы, инкапсуляция, наследование.
- **Современный синтаксис ES6+**: модули, `async/await`, деструктуризация, spread/rest.
- **Продвинутая работа с DOM API** без использования сторонних библиотек.

## 🎨 Часть 2: Профессиональный Веб-дизайн и Верстка (`/web-design-semester-1`)

Этот раздел содержит проекты, демонстрирующие владение HTML и CSS на уровне, необходимом для создания сложных, адаптивных и эстетически выверенных макетов.

### Ключевые проекты и продемонстрированные навыки:

1.  **Резюме, визитки, галереи**: Демонстрация фундаментальных навыков:
    *   **Семантическая верстка (HTML5)** для улучшения доступности и SEO.
    *   Глубокое понимание **CSS Box Model** и позиционирования.
    *   Работа с **кастомными шрифтами, фонами и градиентами**.

2.  **Интерактивные карты и сложные сетки**:
    *   Применение **CSS Flexbox** и **CSS Grid Layout** для создания сложных и адаптивных двумерных макетов.
    *   **Стилизация и анимация SVG** напрямую через CSS.
    *   Создание интерактивных эффектов с помощью псевдоклассов (`:hover`) и **CSS Transitions**.

### Общие компетенции в веб-дизайне:
- **Адаптивная и "резиновая" верстка** с использованием медиа-запросов.
- **Кроссбраузерная совместимость**.
- **Внимание к деталям (Pixel Perfect)** и точное следование визуальным макетам.
- Владение современными техниками для построения сложных сеток.

---

## 🏆 Итог

Эта коллекция проектов подтверждает квалификацию разработчика, способного решать широкий спектр задач: от создания сложной логики и архитектуры веб-приложений на JavaScript до реализации визуально безупречных и адаптивных пользовательских интерфейсов с помощью HTML и CSS.
