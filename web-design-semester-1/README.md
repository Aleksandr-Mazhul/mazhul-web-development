# Обзор Проектов — Веб-дизайн (Семестр 1 и 2)

Данная директория содержит коллекцию проектов, демонстрирующих комплексное владение технологиями HTML и CSS для создания
современных, адаптивных и визуально привлекательных веб-интерфейсов. Работы охватывают как базовые принципы верстки, так
и продвинутые техники стилизации и позиционирования.

Каждый проект представляет собой практическое применение теоретических знаний в области веб-дизайна и
фронтенд-разработки.

## 🚀 Ключевые Проекты и Демонстрируемые Навыки

### 1. **Резюме и визитки (`/Chernuho_resume`, `/kravchenko_resume`, `/business_card`)**

Эти проекты демонстрируют фундаментальные навыки семантической верстки и стилизации контента:

- **Семантический HTML**: Использование тегов (`<header>`, `<footer>`, `<main>`, `<section>`, `<article>`) по их прямому
  назначению для улучшения доступности (Accessibility) и SEO.
- **CSS Box Model**: Глубокое понимание свойств `margin`, `padding`, `border` и `box-sizing` для точного управления
  размерами и отступами элементов.
- **Работа с текстом и шрифтами**: Подключение кастомных шрифтов (`@font-face`), управление размером, начертанием,
  межстрочным и межбуквенным интервалом.
- **Фоновые изображения и градиенты**: Использование `background-image`, `background-size`, `background-position` для
  создания визуально насыщенных блоков.

### 2. **Галерея (`/gallery`)**

Проект галереи показывает умение работать с сетками и создавать сложные многостраничные макеты:

- **Продвинутое позиционирование**: Использование `position: relative/absolute/fixed` для создания нетривиальных
  композиций и наложения элементов.
- **CSS Flexbox**: Применение гибких контейнеров для выравнивания и распределения элементов в рядах и колонках. Этот
  подход используется для создания адаптивных навигационных па��елей и карточных сеток.
- **Псевдоклассы и псевдоэлементы**: Активное использование `:hover`, `:focus`, `::before`, `::after` для добавления
  интерактивности и декоративных элементов без лишней HTML-разметки.

### 3. **Интерактивные карты (`/map`, `/mymap`)**

Эти проекты демонстрируют способность работать с векторной графикой (SVG) и создавать сложные интерактивные интерфейсы
на чистом CSS:

- **Стилизация SVG**: Управление цветами (`fill`, `stroke`), толщиной линий и другими параметрами SVG-элементов напрямую
  из CSS.
- **Интерактивность на CSS**: Создание эффектов при наведении (`:hover`) на отдельные регионы карты, что показывает
  умение работать со сложными вложенными структурами и селекторами.
- **Адативная верстка**: Использование медиа-запросов (`@media`) для адаптации макета под разные размеры экранов (
  например, мобильная и десктопная версии карты).

### 4. **Сложные макеты (`/table`, `/pigfly`)**

Эти работы показывают владение продвинутыми техниками верстки:

- **CSS Grid Layout**: В проекте с таблицей (и других) используется CSS Grid для создания сложных двумерных сеток,
  которые легко адаптируются и управляются. Это демонстрирует владение самой современной технологией для построения
  макетов.
- **Трансформации и анимации**: Применение `transform: translate/rotate/scale` и `transition` для создания плавных
  анимаций и визуальных эффектов, которые делают интерфейс более живым и динамичным.
