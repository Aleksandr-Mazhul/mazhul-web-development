# Картинная галерея Ван Гога

Интерактивная веб-галерея с произведениями известного художника Винсента Ван Гога.

## 📋 Структура проекта

```
gallery/
├── index.html                 # Главная страница с галереей
├── bedroom-in-arles.html      # Спальня в Арле
├── dance-hall.html            # Танцевальный зал
├── first-steps.html           # Первые шаги
├── cafe-at-night.html         # Ночное кафе
├── night-cafe.html            # Ночная терраса кафе
├── white-house-at-night.html  # Белый дом ночью
├── potato-eaters.html         # Едоки картофеля
├── red-vineyards.html         # Красные виноградники в Арле
├── styles.css                 # Единая таблица стилей
├── images/                    # Папка с изображениями картин
└── README.md                  # Этот файл
```

## 🎨 Описание проекта

Проект представляет собой отзывчивую веб-галерею, состоящую из:

- **Главная страница** - сетка картин в две колонки с превью
- **Страницы деталей** - полная информация о каждой картине с навигацией между ними
- **Адаптивный дизайн** - красивая визуализация в стиле картин Ван Гога

## 🚀 Использование

1. Откройте `index.html` в браузере
2. Нажимайте на любую картину для просмотра подробной информации
3. Используйте стрелки навигации для перемещения между картинами
4. Переходите на главную страницу через логотип

## 🎯 Основные особенности

✨ **Семантический HTML**
- Используются правильные теги: `<header>`, `<main>`, `<footer>`, `<article>`, `<section>`
- Структурированный и понятный код

🎨 **Современный CSS**
- Единая таблица стилей `styles.css`
- Используется сетка (CSS Grid) для компоновки картин
- Гладкие переходы и анимации при наведении
- Правильные классы в формате kebab-case

🔗 **Навигация**
- Кольцевая навигация между картинами
- Быстрый переход на главную страницу
- Кликабельные логотипы и кнопки

📱 **Отзывчивость**
- Адаптивный дизайн для всех устройств
- Изображения масштабируются пропорционально

## 📝 Соглашения по кодированию

### Классы CSS
- Используется **kebab-case**: `gallery-card`, `nav-button-left`, `footer-links`
- Смысловые названия: `.gallery-grid`, `.detail-content`
- Структурированные разделы с комментариями

### Атрибуты элементов
- `lang="ru"` на главном элементе html
- Правильные `lang` атрибуты для мультиязычного контента
- `rel="noopener noreferrer"` для внешних ссылок
- `alt` текст для всех изображений

### Структура HTML
- Один `<h1>` на странице (в header)
- Семантические элементы (`<article>`, `<section>`)
- Логичная иерархия заголовков
